Elevate your WooCommerce store's order processing with customized receipts and packing slips, complete with QR and barcodes for efficient tracking.

== Description ==

Seris Order Manager is a powerful plugin designed to enhance your WooCommerce store's order processing. It allows you to effortlessly generate customized receipts and packing slips for orders, complete with QR codes and barcodes, improving both back-end management and customer experience.

== Features ==

* Generate customized receipts and packing slips for WooCommerce orders.
* Include QR codes and barcodes for efficient tracking and verification.
* Seamless WooCommerce integration for back-end and customer workflows.
* Detailed order summaries and packaging details included with each shipment.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seris-receipt-packaging-slip` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > Settings > Woo Receipt and Packaging Slip to configure the plugin settings as needed.
4. Start generating receipts and packaging slips from the WooCommerce orders page.

== Frequently Asked Questions ==

= How do I install the plugin? =
You can install the plugin through the WordPress plugins screen directly or by uploading the files to /wp-content/plugins/ directory and activating it.

= Can I customize the templates for the receipts and slips? =
Yes, you can customize them through settings available in WooCommerce > Settings > Woo Receipt and Packaging Slip.

= Are QR codes and barcodes included by default? =
Yes, the plugin generates QR codes and barcodes for all orders to enhance tracking and verification.
